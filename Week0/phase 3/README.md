🚀 PySpark Data Analysis Practice
📌 About

In this practice, I worked with customers and sales datasets using PySpark.
The goal was to clean data and perform basic analysis like aggregations, joins, and reporting.

🛠️ What I Did
🔹 1. Daily Sales Calculation
Read sales data from CSV
Converted total_amount to numeric
Calculated total sales per day using groupBy() and sum()
🔹 2. City-wise Revenue
Joined customers and sales data
Grouped by city
Calculated total revenue for each city
🔹 3. Repeat Customers
Grouped sales data by customer_id
Counted number of orders
Filtered customers with more than 2 orders
🔹 4. Customer-wise Total Spend
Joined customer and sales data
Created full customer name
Calculated total spend for each customer by city
🔹 5. Final Reporting Table
Joined both datasets
Calculated:
Total spend per customer
Number of orders
Created a final report with customer details
📊 Key Concepts Learned
Reading CSV files in PySpark
Data type conversion using cast()
Aggregations (sum, count)
Joins (combining datasets)
GroupBy operations
Filtering data
