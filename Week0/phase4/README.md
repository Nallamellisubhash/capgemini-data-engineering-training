In this practice, I worked with sales and customers datasets using PySpark.
I performed different operations like aggregation, joins, filtering, and segmentation to understand real-world data analysis.

What I Did
 1. Daily Sales
Calculated total sales for each day using groupBy() and sum()
 2. City-wise Revenue
Tried grouping data by city to calculate total revenue
 3. Customer Spending Analysis
Joined sales and customers data
Created full customer name
Found top 5 customers with lowest spending
 4. Repeat Customers
Counted number of orders per customer
Filtered customers with more than one order
 5. Customer Segmentation
Calculated total spend per customer
Segmented customers into:
Gold (>10000)
Silver (5000–10000)
Bronze (<5000)
 6. Final Reporting Table
Combined all insights into one table
Included:
Customer name
City
Total spend
Order count
Segment

Key Learnings
How to use groupBy() and aggregation functions
How to join multiple datasets
How to create new columns using withColumn()
How to apply conditions using when()
How to build a complete reporting pipeline
