In this task, I worked with order data using PySpark. I performed different operations like aggregation, ranking, window functions, and segmentation to understand how real-world data is analyzed.

What I Did

I calculated total spend for customers and identified top customers in each city. I computed daily sales and created a running total to track growth over time. I analyzed product performance by ranking products within each category.

I also calculated customer lifetime value by finding total spend per customer. Then I segmented customers into Gold, Silver, and Bronze based on their spending. Finally, I combined all the results into one final report.

Key Learnings

I learned how to use groupBy and aggregation functions for analysis. I understood how window functions help in ranking and cumulative calculations. I learned different ways to segment customers and how to build a complete reporting pipeline.

Conclusion

This task helped me understand how to process data, generate insights, and create meaningful reports using PySpark.
