In this task, I worked with a messy dataset and applied data cleaning techniques using PySpark.
The goal was to understand how real-world data can contain errors and how to fix them.

What I Did

Identified null values, duplicates, and invalid data
Removed rows with missing customer_id
Filled missing values in name and city
Removed duplicate records
Filtered invalid age values
Compared row counts before and after cleaning
Performed aggregation (customers per city)
 

Key Learnings
Real-world data is often messy
Data cleaning is very important before analysis
Invalid or missing data can affect results
Validation (before/after count) is necessary
