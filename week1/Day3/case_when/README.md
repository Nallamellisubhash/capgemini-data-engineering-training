# SQL Practice – CASE WHEN

##  Overview

This project demonstrates the use of the **CASE WHEN** statement in SQL for conditional logic and data transformation.

---

## Definition

CASE WHEN is used to apply conditional logic in SQL queries. It allows you to create new columns or categorize data based on specific conditions.

---

## 🔧 Purpose

* Perform conditional transformations
* Categorize data into groups
* Apply business logic inside queries
* Improve readability of complex conditions

---

##  How It Works

CASE WHEN evaluates conditions one by one:

* If a condition is TRUE → corresponding result is returned
* If no condition matches → ELSE part is executed

---

##  Key Concepts

* Works like **IF-ELSE** in programming
* Can be used in SELECT, WHERE, ORDER BY
* Supports multiple conditions
* Can be combined with aggregate functions

---

## Use Cases

* Categorizing salary (High, Medium, Low)
* Classifying sales performance
* Creating custom labels for reports
* Handling missing or special values
* Conditional aggregation

---

##  Types of CASE WHEN

* Simple CASE → compares a column with values
* Searched CASE → uses conditions (more flexible)

---

##  Important Points

* Conditions are checked in order
* First matching condition is executed
* ELSE is optional but recommended
* Improves query readability

---

##  Summary

CASE WHEN is a powerful tool for adding logic inside SQL queries, making data analysis more flexible and meaningful.
