

## About

In this task, I worked with customer and order datasets that contained messy and invalid data. I performed data cleaning, validation, transformations, and analysis using PySpark.

## What I Did

I created sample datasets with null values, duplicates, and incorrect records. Then I cleaned the data by removing nulls, fixing negative values, and trimming unwanted spaces.

I validated the data by identifying invalid customer IDs using left_anti join. After that, I joined the datasets and calculated total spend and order count for each customer.

I used window functions to rank customers based on their spending and also identified the top 3 customers in each city.

I performed date-based analysis by extracting month from order dates, calculating monthly sales, and finding date differences. I also did trend analysis by comparing current month sales with previous month sales.

## Key Learnings

I learned how to clean real-world data and handle null and invalid values. I understood how joins help in combining datasets and validating data. I learned how to use window functions for ranking and analysis.

I also learned how to perform time-based analysis like monthly aggregation and trend calculation.

## Conclusion

This task helped me understand the complete data processing pipeline from cleaning to analysis using PySpark.
