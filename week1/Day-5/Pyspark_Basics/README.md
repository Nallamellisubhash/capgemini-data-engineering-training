#  DAY 5 – PySpark Data Processing & Transformations

##  Overview
This project demonstrates the use of **PySpark** for data processing, transformation, and analysis. It focuses on working with structured data using DataFrames and applying various transformations to clean and manipulate the data.

The project highlights how PySpark can be used in real-world data engineering workflows.

---

##  Objectives
- Understand PySpark DataFrame operations
- Perform data cleaning and transformation
- Work with built-in PySpark functions
- Apply filtering, aggregation, and column operations
- Prepare data for analysis

---

##  Technologies Used

- PySpark 


---

##  Project Description
In this project, data is loaded into PySpark DataFrames and processed using various transformations.

Key tasks include:
- Reading data from files (CSV/other formats)
- Handling missing values
- Filtering records based on conditions
- Creating new columns
- Applying transformations on existing columns
- Performing aggregations and group operations

---

##  Key Concepts Covered

###  Data Loading
- Reading CSV files into DataFrames
- Inferring schema and handling headers

---

###  Data Cleaning
- Handling null values
- Removing duplicates
- Standardizing data formats

---

###  Column Transformations
- Creating new columns using `withColumn()`
- Applying expressions and functions
- Renaming columns

---

###  Filtering Data
- Using conditions to filter rows
- Applying multiple filters

---

###  Aggregations
- Grouping data using `groupBy()`
- Applying aggregate functions like sum, count, avg

---

###  Built-in Functions
- String functions
- Numeric functions
- Date functions
