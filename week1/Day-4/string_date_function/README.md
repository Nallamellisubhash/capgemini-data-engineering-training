#  SQL Functions Project – String, Date & Numeric Functions

##  Overview
This project demonstrates the usage of **SQL functions** across three major categories:
- String Functions 
- Date Functions 
- Numeric Functions   

These functions are essential for **data transformation, cleaning, and analysis** in real-world database systems.

The project focuses on how different types of functions can be applied to structured data to extract meaningful insights and improve data quality.

---

##  Objectives
- Understand different categories of SQL functions
- Learn how to manipulate text, date, and numeric data
- Perform data cleaning and transformation
- Build a strong foundation for SQL-based data analysis

---

##  Technologies Used
- SQL 
- Relational Databases (MySQL / PostgreSQL / SQL Server)

---

## Project Description
This project works on a dataset containing various types of data such as:
- Text values (names, cities, etc.)
- Date values (joining date, order date, etc.)
- Numeric values (salary, amount, etc.)

Using SQL functions, the project demonstrates:
- Data formatting and transformation
- Data cleaning techniques
- Extracting useful information from raw data

---

##  Key Concepts Explained

###  String Functions
String functions are used to manipulate and format textual data.

**Concepts Covered:**
- Changing text case (uppercase/lowercase)
- Extracting parts of strings
- Measuring string length
- Removing extra spaces
- Combining multiple text values
- Replacing characters or words

---

###  Date Functions
Date functions help in handling and analyzing date and time data.

**Concepts Covered:**
- Extracting year, month, and day
- Formatting date values
- Performing date calculations
- Finding differences between dates
- Working with current date and time

---

###  Numeric Functions
Numeric functions are used to perform mathematical operations and calculations.

**Concepts Covered:**
- Rounding values
- Performing arithmetic operations
- Calculating absolute values
- Finding averages, sums, and other aggregations
- Controlling decimal precision

---

##  Real-World Applications

- Preparing data for dashboards and reports
- Performing financial and business calculations
- Handling date-based analysis (sales trends, timelines)
