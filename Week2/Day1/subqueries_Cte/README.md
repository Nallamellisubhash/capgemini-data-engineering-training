#  SQL CTE & Subquery Practice Project

##  Project Overview
This project focuses on practicing SQL concepts including subqueries, Common Table Expressions (CTE), joins, and aggregations. It demonstrates real-world query scenarios using sample datasets like employees, departments, and orders.

---

##  Technologies Used
- SQL (Standard SQL / MySQL-like syntax)
- Relational Database Concepts

##  Common Table Expressions (CTE)

### What is CTE?
A CTE (Common Table Expression) is a temporary result set used to simplify complex queries and improve readability.

### CTE Use Cases
- Filtering high-salary employees  
- Department-wise average salary  
- Joining employee and department data  
- Aggregating order data  
- Comparing employee salary with department average
